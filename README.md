# Calculadora Simples com Interface Gráfica

Uma calculadora simples desenvolvida em Java que demonstra conceitos básicos de programação orientada a objetos e interação com usuário através de interface gráfica.

## 📋 Sobre o Projeto

Este projeto implementa uma calculadora que permite realizar operações matemáticas básicas, incluindo:
- Soma
- Subtração
- Operações com número preferido do usuário

O programa utiliza `JOptionPane` para criar uma interface gráfica simples de diálogo com o usuário.

## 🚀 Funcionalidades

- Entrada de dados via caixas de diálogo gráficas
- Armazenamento de um número preferido do usuário
- Operações de soma e subtração
- Método especial para somar qualquer número com o número preferido
